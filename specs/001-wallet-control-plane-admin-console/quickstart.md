# Quickstart: Wallet Control Plane Admin Console

## Prerequisites

- Node.js 20 or later.
- npm.

## Install

```bash
npm install
```

## Run The Demo

```bash
npm run dev
```

Open the local URL printed by Vite. The app uses local mock JSON only and does
not require secrets or backend services.

## Validate

```bash
npm test
npm run build
```

## Screenshot Flow

1. Start the dev server.
2. Capture the Dashboard at desktop width.
3. Capture Policies with a policy selected.
4. Capture Pending Approvals with the transaction detail open.
5. Capture Audit Log at mobile width.

All screenshots must preserve visible mock/reference labeling.
